0x01 Algorithm

Leet Code #804 Unique Morse Code Words

Soulution:
本题一开始很自然想到用字符映射和列表去重，但使用列表索引会有更高的效率。

java:

class Solution {
    public int uniqueMorseRepresentations(String[] words) {
        String[] MORSE = new String[]{".-","-...","-.-.","-..",".","..-.","--.", "....","..",".---","-.-",".-..","--","-.", "---",".--.","--.-",".-.","...","-","..-", "...-",".--","-..-","-.--","--.."};
        Set<String> seen = new HashSet();
        for (String word: words) {
            StringBuilder code = new StringBuilder();
            for (char c: word.toCharArray())
                code.append(MORSE[c - 'a']);
            seen.add(code.toString());
        }
        return seen.size();
    }
}

python:

class Solution:
    def uniqueMorseRepresentations(self, words: List[str]) -> int:
        k = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k', 'l', 'm', 'n', 'o', 'p', 'q', 'r', 's', 't', 'u', 'v', 'w', 'x', 'y', 'z']
        v = [".-","-...","-.-.","-..",".","..-.","--.","....","..",".---","-.-",".-..","--","-.","---",".--.","--.-",".-.","...","-","..-","...-",".--","-..-","-.--","--.."]
        mp = dict(zip(k, v))
        s = set([''.join([mp[e] for e in wd]) for wd in words])
        return len(s)

0x02 Review

来自Medium的一篇关于键盘的文章
How to write code you will love in the future

很有意思的一篇文章，科研人员专门发明了法语键盘，相较我们常用的键盘（英文键盘，已是事实上的标准）增加了法语字符的输入，如：œ 、 É等。可以想象，这些定制化的键盘可以提高法语输入的效率。其实我想了，我们在日常行为模式已经固定的情况下，能不能再多思考一下，定制化一下适合个人的准则并固化，极大提高个人的效率，不管从生活还是工作方面。如果我们每个人都有一套自己的”定制化“模式，我们整个社会的发展就会提高很大一步。

0x03 Tip

区分一下同步异步和阻塞非阻塞

同步异步：
简单来说，同步是一种可靠的有序运行机制，当我们进行同步操作时，后续的任务是等待当前调用返回，才会进行下一步；而异步则相反，其他任务不需要等待当前调用返回，通常依靠事件、回调等机制来实现任务间次序关系。

阻塞和非阻塞：
在进行阻塞操作时，当前线程会处于阻塞状态，无法从事其他任务，只有当条件就绪才能继续，比如 ServerSocket 新连接建立完毕，或数据读取、写入操作完成；而非阻塞则是不管 IO操作是否结束，直接返回，相应操作在后台继续处理。
